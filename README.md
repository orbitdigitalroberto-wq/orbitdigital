# Orbit Digital — Site (GitHub Pages)

Site one-page (preto + roxo), pronto para publicar no GitHub Pages.

## Personalizar

### WhatsApp
Abra `assets/js/main.js` e troque:

const WHATSAPP_NUMBER = 'SEUNUMEROAQUI';

Exemplo:

const WHATSAPP_NUMBER = '5511999999999';

## Publicar no GitHub Pages

1. Crie um repositório no GitHub (ex.: `orbit-digital`).
2. Envie todos os arquivos desta pasta.
3. No repositório: **Settings** → **Pages**.
4. Em **Branch**, selecione `main` e a pasta `/ (root)` → **Save**.
5. Aguarde o link do seu site aparecer em Pages.

## Ajustes rápidos

- Cores/estilo: `assets/css/style.css`
- Textos: `index.html`
